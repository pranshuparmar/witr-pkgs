# witr-pkgs

> **Why is this running?**
>

`witr` answers this question by explaining the causal chain behind running processes.

For more details about `witr`, visit the [main repository](https://github.com/pranshuparmar/witr).

This repository hosts the self maintained package manifests. `witr` is available on many other community maintained package managers as well.
